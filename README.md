# 🎓 AI Academic Intelligence System

An AI-driven academic performance prediction and advisory system using:

- Gradient Boosting Regression
- KMeans Clustering
- Feature Scaling
- Conversational UI (Streamlit)

## 🚀 Features

- Predict CGPA (10-scale)
- Student segmentation (High Achiever / Moderate / At-Risk)
- Explainable AI (Top Influencing Features)
- Improvement simulation
- Context-aware conversational interface

## 🛠 How to Run

1. Install dependencies:

pip install -r requirements.txt

2. Train model:

python preprocess.py

3. Run app:

streamlit run app.py
